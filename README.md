# CoruñaTech 🌊

> A comunidade tecnolóxica da Coruña, nun só lugar

[![Deploy](https://github.com/corunatech/corunatech.org/actions/workflows/deploy.yml/badge.svg)](https://github.com/corunatech/corunatech.org/actions/workflows/deploy.yml)
[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](./LICENSE)

**CoruñaTech** é unha plataforma que reúne todas as comunidades tecnolóxicas da Coruña e os seus eventos nun único lugar. Descubre meetups, charlas, talleres e conecta coa comunidade tech local.

🌐 **[corunatech.org](https://corunatech.org)**

## ✨ Características

- 🏘️ **Comunidades Tech**: Descubre todas as comunidades tecnolóxicas activas na Coruña
- 📅 **Calendario de Eventos**: Nunca te perdas un meetup, charla ou taller
- 🤖 **Xestión Automatizada**: Engade comunidades e eventos a través de issues

## 🚀 Contribuír

Queres engadir a túa comunidade ou evento? Reportar un bug? Suxerir melloras?

👉 **Le a nosa [Guía de Contribución](./CONTRIBUTING.md)**

Todas as contribucións son benvidas: comunidades, eventos, código, deseño, documentación... Ti decides como axudar!

## 🛠️ Tecnoloxías

Este proxecto está construído con:

- [Astro](https://astro.build) - Framework web
- [TypeScript](https://www.typescriptlang.org/) - Linguaxe
- [GitHub Actions](https://github.com/features/actions) - CI/CD e automatización
