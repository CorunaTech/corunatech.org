# Guía de Contribución

Benvido/a a CoruñaTech! 🎉

Grazas polo teu interese en contribuír á nosa plataforma. CoruñaTech é unha comunidade aberta que dá a benvida a contribucións de todo tipo.

## Como Podes Contribuír

### 🏘️ Engadir a Túa Comunidade

Se formas parte dunha comunidade tecnolóxica da Coruña, podes engadila á plataforma:

1. Vai a [Issues](../../issues/new/choose) e selecciona **"Engadir Comunidade"**
2. Completa o formulario cos datos da túa comunidade
3. Unha Pull Request crearase automaticamente
4. Os membros da comunidade CoruñaTech revisarán e aprobarán manualmente a solicitude

⚠️ **Importante**: As comunidades deben estar relacionadas co ámbito tecnolóxico e ter presenza na Coruña. A revisión é manual para asegurar a calidade e pertinencia do contido.

### 📅 Engadir un Evento

Podes engadir eventos organizados polas comunidades rexistradas en CoruñaTech:

1. Vai a [Issues](../../issues/new/choose) e selecciona **"Engadir Evento"**
2. Completa o formulario cos detalles do evento
3. Unha Pull Request crearase automaticamente
4. Os mantedores revisarán e aprobarán a solicitude
5. Ao fusionarse, o evento publicarase automaticamente na web

⚠️ **Importante**: Só se poden publicar eventos organizados por comunidades que xa forman parte de CoruñaTech. Se a túa comunidade aínda non está rexistrada, engádea primeiro seguindo os pasos anteriores.

💡 **Consello**: Segue coidadosamente as instrucións e formatos indicados nos formularios, xa que o proceso está automatizado e erros nos datos poden impedir que a túa solicitude se procese correctamente.

### 🐛 Reportar Bugs

Atopaches un erro na web? Axúdanos a mellorala:

1. Vai a [Issues](../../issues/new/choose) e selecciona **"Reporte de Bug"**
2. Describe o problema cos detalles solicitados no formulario

### 💡 Suxerir Funcionalidades

Tes ideas para mellorar CoruñaTech?

1. Vai a [Issues](../../issues/new/choose) e selecciona **"Solicitude de Funcionalidade"**
2. Explica a túa proposta seguindo o formulario

### 💻 Contribuír ao Código ou Deseño

As contribucións técnicas e de deseño son sempre benvidas! Se queres mellorar o código, o deseño ou a funcionalidade da plataforma:

1. Revisa as [issues abertas](../../issues) para ver se hai algo no que queiras traballar
2. Fai un fork do repositorio
3. Crea unha rama para os teus cambios
4. Implementa as túas melloras
5. Crea unha Pull Request cunha descrición clara dos cambios

Se tes dúbidas ou queres discutir unha idea antes de implementala, abre unha issue primeiro!

## Código de Conduta

Este proxecto adhírese ao [Contributor Covenant Code of Conduct](./CODE_OF_CONDUCT.md). Ao participar, espérase que respectes este código.

En resumo, comprométeste a:

- 🤝 Tratar a todas as persoas con respecto
- 💬 Manter unha comunicación constructiva e profesional
- 🌈 Fomentar un ambiente inclusivo e acolledor
- 🎯 Centrarte no que é mellor para a comunidade tecnolóxica

Por favor, le o [código completo](./CODE_OF_CONDUCT.md) para máis detalles. Se observas comportamentos inaceptables, repórtalos abrindo unha issue.

## Preguntas?

Se tes dúbidas sobre o proceso de contribución:

1. Revisa a [documentación](https://corunatech.org)
2. Consulta as [issues pechadas](../../issues?q=is%3Aissue+is%3Aclosed) para ver exemplos
3. Abre unha issue cunha pregunta
